# Проект: Прогнозирование стоимости страховых выплат

## Описание проекта

Цель проекта - разработать модель машинного обучения для предсказания итоговой стоимости страхового возмещения `UltimateIncurredClaimCost` на основе характеристик работника, описания страхового случая и начальной оценки стоимости.

## Датасет

Используется набор данных **Workers Compensation** из OpenML, ID: `42876`. Датасет содержит 100 000 записей, 13 признаков и одну целевую переменную.

## Установка и запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Структура проекта

- `app.py` - точка входа Streamlit-приложения с навигацией `st.navigation` и `st.Page`.
- `analysis_and_model.py` - основная страница анализа, обучения моделей и прогноза.
- `presentation.py` - страница с презентацией проекта через `streamlit-reveal-slides`.
- `model_utils.py` - общая логика загрузки данных, предобработки, обучения и оценки моделей.
- `.streamlit/config.toml` - светлая тема приложения с акцентным цветом Tiffany.
- `reports/` - сформированный отчет, таблицы и графики.
- `scripts/` - служебные скрипты генерации отчета и материалов.
- `video/demo.mp4` - короткая демонстрация интерфейса приложения.
- `video/videocapture.mp4` - видеозапись работы приложения.

## Используемые модели

В проекте сравниваются Linear Regression, Ridge Regression, Random Forest Regressor и XGBoost Regressor. Для оценки используются MAE, MSE, RMSE и R2.

## Функционал приложения

- анализ структуры датасета и статистики признаков;
- обучение моделей и вывод метрик качества;
- сравнение графиков моделей: факт-прогноз, остатки и абсолютные ошибки;
- анализ важности признаков;
- расчет прогноза для нового страхового случая.
